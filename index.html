The trouble here really is taking care of the extras spaces that appear/disappear when the nav is fixed. To do this, we need to know the height of the nav, and the height of the site-wrap. We can get these values by adding the height of the nav to the height of the site-wrap. This is the height of the space between the nav and the site-wrap. We can then use this value to set the top of the site-wrap. The end result is that the site-wrap will always be at the top of the page, and the nav will always be fixed. Sweet!
